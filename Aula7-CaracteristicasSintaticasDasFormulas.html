Características Sintáticas das Fórmulas


primeiramente, descobriremos como se calcular o comprimeiro de uma fórmula: basta contar seus símbolos como no exemplo

agora iremos definir como funciona o comprimento de uma fórmula recursivamente, para podermos aplicar na prática:
1.Se H = P̂, então comp[H] = 1;
2.Comp[¬H] = comp[H + 1];
3.Comp[H v G] = comp[H] + comp[G] + 1;
4.Comp[H ^ G] = comp[H] + comp[G] + 1;
5.Comp[H → G] comp[H] + comp[G] + 1;
6.Comp[H ←→ G] = comp[H] + comp[G] + 1;

Referência: Souza, J.N., Lógica Para Ciência da Computação e Áreas Afins, Editora Campus, 3a ed., 2015. Tema da aula: Capítulo 1, página 10.
